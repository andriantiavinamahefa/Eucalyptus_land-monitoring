# Eucalyptus_land-monitoring
This repo reports my training in Land monitoring. For this case, i assess the evolution of our Eucalyptus land over years from 2021-2026 using Google Earth Engine (the scrpit is below in the file script used in GEE), via Sentinel-2 images. After having the dataset accross the time range, the data has been exported in drive to be used in python after (script is below in the file named script named "Script used to generate the evolution of NDVI..."). The file NDVI_our_land is the entire dataset. The ndvi_evolution.png shows the final result, indicating that the highest NDVI we have got was back in 2021. After that, deforestation has been heavily practiced almost each year leading to growth limitation.
